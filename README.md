**Project Title

Tic Tac Toe AI (C++ Project)

Project Overview

This project is a Tic Tac Toe game with an AI opponent built in C++. The AI uses the Minimax algorithm with recursion and backtracking to make optimal moves.

Goal: Create a game where the AI is unbeatable and the player can practice strategy against it.

Scope: Console-based game, easy to run on any machine with C++ support.

Key Features

Unbeatable AI:

Uses Minimax algorithm to evaluate all possible moves.

AI always plays the optimal move, making it very challenging for the player.

Recursion & Backtracking:

The AI explores all potential game states recursively.

Backtracking ensures it can revert moves and find the best outcome for each turn.

User-Friendly Console Interface:

Players can input moves easily.

The board updates after each move with a clear display.

Win/Draw Detection:

The game detects wins, losses, or draws automatically.

Technical Skills Demonstrated

C++ Programming: Loops, arrays, functions, and input/output handling.

Algorithms: Minimax algorithm for decision making.

Recursion & Backtracking: Core concepts applied to game AI.

Problem-Solving: Designing a strategy-based AI.

GitHub Workflow: Uploading, version control, and project documentation.**
